## Pequeña aplicacion web

La aplicacion web creada en esta seccion fue hecha con apoyo de librerias externas para facilitar la programacion funcional y el manejo de estilos.
Para javascript se utilizo la libreria R y para css, se utilizo Tachyons CSS.
Es una aplicacion que muestra una tabla con dos columnas, una para comida y otra con el valor calorico.
Al final se totaliza la cantidad de calorias
